Homebrew Tap for AgenC
======================

This is the official [Homebrew](https://brew.sh/) tap for [AgenC](https://github.com/mieubrisse/agenc), an agent orchestration system.

Installation
------------

```bash
brew tap mieubrisse/agenc
brew install agenc
```

Updating
--------

```bash
brew update
brew upgrade agenc
```

About AgenC
-----------

AgenC is an agent orchestration system. For documentation, issues, and source code, visit the main repository:

**https://github.com/mieubrisse/agenc**
